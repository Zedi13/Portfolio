Projet Sonde méteo

Récupération des données d'une sondre méteo

Notre projet consistait à afficher les informations météorologiques d'une sonde sur un site web en utilisant une API pour récupérer les données en temps réel. Cette intégration nous a permis d'afficher les données météo actualisées de manière dynamique, offrant aux utilisateurs un accès rapide et précis aux prévisions météorologiques.
